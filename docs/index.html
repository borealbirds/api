Boreal Birds JSON API

Then JSON API hosts summary results from thead Boreal Avian Modelling Project. Results include static images and data in JSON (JavaScript Object Notation) format that can be consumed by a wide array of modern programming languages.

The API base URL is https://borealbirds.github.io/api/

Versions

The API starts with v4, no previous version are hosted here. The version path ./v4/ gives a JSON array of the species whose results are available

Species

The species path within version (e.g. ./v4/species/ALFL/ gives a JSON array of the results for that species.
